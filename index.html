ColorTrompon a Ceylon syntax highlighter

ColorTrompon adds syntax highlighting to a Ceylon code snippet, based purely on lexical analysis.

This fork of Lucas Werkmeister's ColorTrompon serves as a demo app for the Dart backend for Ceylon.

It has been rewritten in 100% Ceylon (see the code below), compiled to Dart, and then compiled to JavaScript using dart2js.

Modules used by this app:

  • ceylon.language (api)
  • ceylon.collection (api)
  • ceylon.html (api)
  • ceylon.lexer.core (github)
  • ceylon.interop.dart
  • dart.core (api)
  • dart.async (api)
  • dart.html (api)
